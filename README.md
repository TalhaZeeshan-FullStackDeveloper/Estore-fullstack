# 🛒 Ecommerce Platform — Full Stack Web Application

A modern and scalable **Full Stack Ecommerce Platform** developed using **Laravel, PHP, MySQL, JavaScript, HTML, CSS, and Bootstrap**.

This project provides a complete online shopping experience for customers along with a secure and feature-rich administration panel for managing the business efficiently.

---

# ✨ Project Overview

This platform was designed to deliver a complete ecommerce workflow including:

🛍 Product Browsing
🛒 Cart Management
📦 Order Processing
💳 Checkout Experience
📈 Admin Operations
📋 Invoice Management
🔐 Authentication & Security

# ⚙ Installation & Setup

### Clone Repository

```bash
git clone https://github.com/talha1122146456/Ecommerce.git
cd Ecommerce
```

### Install Dependencies

```bash
composer install
npm install
```

### Configure Environment

```bash
cp .env.example .env
```

Update database credentials.

---

### Generate Application Key

```bash
php artisan key:generate
```

---

### Run Migration

```bash
php artisan migrate
```

---

### Start Development Server

```bash
php artisan serve
```

Open:

```plaintext
http://127.0.0.1:8000
```
